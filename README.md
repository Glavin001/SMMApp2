## Installation

### Global Dependencies

#### Node.js and NPM

See http://nodejs.org/

##### Bower

See http://bower.io/

##### Grunt

See http://gruntjs.com/

#### Ruby & Compass

See http://compass-style.org/install/

#### Dependencies

```bash
npm install
bower install
```

## Usage

### Run the front-end web app

```bash
grunt serve
```

### Run the server

```bash
node server/
```
